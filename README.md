# mahmoud.jp

Personal portfolio and writing space. Bilingual (English / 日本語), built with Next.js.

🌐  **Live:** [mahmoud.jp](https://mahmoud.jp)

## What's inside

- **Home** — about, experience, skills, languages, freelance work, contact
- **/projects** — featured work, including DTP Master
- **/uses** — daily software and hardware
- **/writing** — long-form notes

## Stack

- [Next.js 15](https://nextjs.org) (App Router) · TypeScript · Tailwind CSS
- [Resend](https://resend.com) for the contact form
- Bilingual EN/JA via React Context + localStorage
- JSON-LD `Person` schema for SEO
- Deployed on [Vercel](https://vercel.com)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

To enable the contact form, set `RESEND_API_KEY` in `.env.local`.
